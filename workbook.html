What's inside

Five parts, cover to cover

Not a stack of blank practice pages — a proper learning book with real instruction, worked examples, projects, and answer keys.

1
Tiny Markdown basics

What Markdown is, where it's used, why plain text matters — and why this is emphatically not "coding." The gentle on-ramp for total beginners.

2
The lessons

One chapter per lesson. Every chapter follows the same reliable rhythm:

what the mark does the raw Markdown the rendered result one common mistake three practice tasks try this online →
3
Real-life Markdown projects

Where the book earns its place — guided projects that turn syntax into finished, useful documents:

• Write a README
• Draft a product description
• Format a blog post
• Write meeting notes
• Make a recipe card
• Build a small documentation page
• Create a task list
• Prepare content for a static site
4
Cheat sheets & troubleshooting

A tear-out quick reference — table syntax, image syntax, links, code blocks — plus a "why didn't this render?" section for the mistakes everyone makes (looking at you, tables and missing blank lines).

5
The final project

The printed version of the course's final exam: write a small article using every Markdown skill, then test it live on the playground. Answer keys included, so you can check your own work.
