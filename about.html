About

About this project

Markdown Course is a free, interactive course that teaches Markdown — the simple formatting language behind README files, chat apps, note tools, and more — from the very first # to a full document.

Why we built it

Most Markdown guides are walls of reference text. We wanted the opposite: short lessons, a live editor, instant feedback, and a little fun along the way. You learn by doing — typing real Markdown and watching it render — rather than by reading about it.

What you'll learn

Ten bite-sized lessons cover headings, bold and italic, lists, links, inline code, code blocks, blockquotes, images, task lists, and dividers — followed by a final exam and a printable certificate. Each lesson includes a tip, an optional bonus challenge, and a quick quiz.

Who it's for

Total beginners, students, writers, and anyone who has ever stared at a text box wondering how to make a word bold. No coding background required.
